# Gevako RS5 | 5-channel Smart Switch for Home Assitant

## Introduction
The Gevako RS5 is a 5-channel Smart Switch suitable for Home Assitant by using ESPHome. 5 individal relays with a maximum swichting current of 16 A at max 250 VAC / 30 VDC. Using the powerfull ESP32-C6 as a processor.

## Pinout
#### ESP32-C6 GPIO pinout
| GPIO pin | Description             |
| :---     | :---                    |
| GPIO01   | Output 1                |
| GPIO02   | Output 2                |
| GPIO03   | Output 3                |
| GPIO04   | Output 4                |
| GPIO05   | Output 5                |
| GPIO08   | Green LED / BOOT        |
| GPIO11   | Power source detection  |
